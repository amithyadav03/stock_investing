=== MANDATORY CHAIN OF THOUGHT — FOLLOW EACH STEP EXACTLY ===

STEP 1 — TECHNICAL THESIS (chain_of_thought_1_technicals):
- Identify the dominant chart structure: is price trending, consolidating, or reversing?
- Check RSI: below 30 = oversold bounce, above 70 = overbought short.
- Check MACD histogram: positive and rising = bullish momentum; negative = bearish.
- Identify key support/resistance levels from the chart image.
- Conclude if the technical setup is valid for a swing trade.

STEP 2 — FUNDAMENTAL VALIDATION (chain_of_thought_2_fundamentals):
- Does the macro environment ({macro_env}) support this direction?
- If BEARISH macro and you are considering BUY: significantly reduce conviction.
- Factor in the sentiment score. Does news confirm or contradict the technical thesis?

STEP 3 — RISK PROFILE (chain_of_thought_3_risk):
- Set Stop Loss = entry - (2 × ATR). This is NON-NEGOTIABLE.
- Set Take Profit = at minimum 1:2 risk/reward from entry.
- Risk Percentage: default 5% of capital. Reduce to 3% if macro is BEARISH.

STEP 4 — TIMEFRAME ESTIMATION (expected_holding_days):
- A swing trade typically lasts 5–30 days.
- Estimate based on: distance from entry to target ÷ average daily ATR movement.
- Short swing (momentum breakout): 5–10 days.
- Medium swing (trend continuation): 10–20 days.
- Positional swing (deep value setup): 20–45 days.
- Output a single integer representing your best estimate in calendar days.

=== OUTPUT ===
Populate all chain_of_thought fields first, then output the final trade decision.
If no valid setup exists, set proposed_action to 'HOLD'.
